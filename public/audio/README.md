# Archivos de Música de Fondo

Coloca aquí los archivos de música de fondo para cada área de enfoque.

## Archivos necesarios

| Archivo | Área | Descripción |
|---------|------|-------------|
| `physical.mp3` | Físico | Sonidos de naturaleza, frecuencia 528Hz, ritmo suave |
| `economic.mp3` | Económico | Tonos cálidos, melodías ascendentes de abundancia |
| `spiritual.mp3` | Espiritual | Cuencos tibetanos, om, atmósfera etérea |
| `energetic.mp3` | Energético | Frecuencias de chakras, cristales, flujo energético |

## Especificaciones recomendadas

- **Formato**: MP3 (320kbps recomendado)
- **Duración**: 15-30 segundos (se reproducirá en loop)
- **Estilo**: Ambient, meditativo, sin letra
- **Volumen**: Normalizado para mezcla suave con narración

## Fuentes de música libre de regalías

- [Pixabay Music](https://pixabay.com/music/)
- [Free Music Archive](https://freemusicarchive.org/)
- [Uppbeat](https://uppbeat.io/)
- [Mixkit](https://mixkit.co/free-stock-music/)

## Nota

Si los archivos no existen, la meditación funcionará sin música de fondo.
